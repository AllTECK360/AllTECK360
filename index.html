Política de Privacidad

Última actualización: [15 de septiempre 2024]

Esta app está diseñada únicamente para interactuar con dispositivos Arduino y no recopila ni comparte datos personales de los usuarios.

Recopilación de Datos

Nuestra aplicación no recopila, almacena ni comparte información personal de ningún tipo. La aplicación únicamente interactúa con dispositivos Arduino para controlar hardware externo, sin involucrar la recolección de datos sensibles.

Permisos

La aplicación puede solicitar permisos para conectarse Wi-Fi con el fin de operar el Arduino de manera remota. Estos permisos son necesarios para la funcionalidad básica de la app y no se usan para otro propósito.

Seguridad

Nos comprometemos a garantizar que la información en el dispositivo del usuario esté segura. La app no almacena ni transfiere información del usuario a servidores externos.

Cambios a esta Política

Nos reservamos el derecho de actualizar esta política en cualquier momento. Se recomienda que los usuarios revisen esta política periódicamente para estar informados de cualquier cambio.

Contacto

Si tienes alguna pregunta sobre esta Política de Privacidad, puedes contactarnos en: [alltecksystems@gmail.com].
